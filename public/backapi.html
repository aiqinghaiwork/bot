1.登录 url：/user/loginin 请求方式：get 请求参数： 
    {
        username: 'yonghuming',
        password: '123456'
    }
响应数据： 
    {
        code: 200,
        msg: "登录成功"
    }
    {
        code: 201,
        msg: "密码或用户名不正确"
    }
2.登出 url：/user/loginout 请求方式：get 请求参数：
响应数据： 
            {code: 200, msg: '登出成功'}
3.获取用户信息 url：/admin/player 请求方式：get 请求参数： 
    {
        lastid: 'sdfafdsfadsxcwe', // 当前页的开始id或结束id
        isnext: 0, // 0表示请求下一页，1表示请求上一页
    }
    第一次进入页面不传lastid，会返回最近的20条数据
响应数据： 
        {code: 200, msg: 'success', rows: players}
4.删除用户信息 url：/admin/player/del 请求方式：get 请求参数： 
    {
        id: 'sdfsfsafs' // 用户id
    }
响应数据： 
        {code: 200, msg: '删除用户成功'}
        {code: 201, msg: '用户不存在'}
